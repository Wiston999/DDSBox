DDSBox
======
DDSBox is a real-time file sharing distributed system.  It has been developed by Olmo Jiménez Alaminos and Víctor Cabezas Lucena as a bachelor's degree final project.

DDSBox allows to share folders and their contents between different users using the DDS Middleware.

DDSBox has been developed using RTI Connext DDS Software. It's mandatory to have a license file to be able to compile a run the application.

## Building the Application ##

*cd* into the root directory and execute:

    gradlew build

or

    gradlew.bat build

The executable will be created at the bin directory.
